This is our DBMS project which is on the topic of CAB BOOKING SERVICE.

In this project, we are making a database for storing the cab, Driver, and passenger information in SQL language and MySQL workbench. We have Django as the backend framework in making this.

The features of the project are:

  1. The passenger and Driver can make their account using their respective email, which is done on the signup page.
  2. Then there is a login page to log in as a passenger or Driver.
  3. There will be a home screen on which we can book a cab by entering pickup and destination location.
  4. The next page will be the payment page, on which there will be the amount, Cab, and Driver information. There will be two buttons on the payment page -- make payment and Cancel cab. The Cancel Cab button will cancel the cab booking and redirect the user to the Home screen.
  5. By clicking on make the payment, we will go to the next page, which will need card details to make payment.
  6. The last page will be a feedback page for the passenger on which the passenger can give feedback about the Driver.
